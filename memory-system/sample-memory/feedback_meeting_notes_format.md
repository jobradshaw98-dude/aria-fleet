Capture meetings as three sections: Decisions, Action items (with owners + dates), Open questions. Never a raw transcript.

**Why:** A transcript hides what actually has to happen.
**How to apply:** Default any meeting summary to this shape. Consistent with [[feedback_response_brevity]].
